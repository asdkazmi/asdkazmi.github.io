Add Classes to Child Elements Using Attribute

writeData("main_data['title']") provide an option to add classes to child elements using it's attribute. This is very useful attribute when the data is dynamic and want to add classes to it's child elements with many options.

To add HTML class to child elements, add attribute data-child and then write classes inside it. writeData("main_data['title']") provide options to add classes to child elements. Here is a simple example to add class to it's child element

Options

writeData("main_data['title']") provide different options to add classes to it's child elements. These options are below

Add Class to Selected Child Elements

To add class to selected child elements, i.e. child elements by own choice, use suffix -nthChd(m,n,...) for each class inside data-child attribute where m,n are integers starting from 0,1,2,....
