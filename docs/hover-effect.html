Hover Effect Attribute

writeData("main_data['title']") an attribute to add classes on hover of element.

To add a class on hover add attribute data-hover and then each class written in this attribute will be added on hover of element.

How to use Hover Classes?

As every class of writeData("main_data['title']") is written in the format of Type-Value-Effect where hover is an effect. Write every class in this format and dont use - while writting any Type or Value because this is a serparator of writeData("main_data['title']") classes. On hover of element, data-hover found a class with same type which we want to add on hover and then replace it with hover class while adding your value.

See here to understand

So, when there will be two classes of same type one before hover and another on hover, then hover class will override first class. This is one main purpose to define every writeData("main_data['title']") class in the format of Type-Valu-Effect. So when developers create there own classes, it's suggested that create own classes in this format for to use writeData("main_data['title']") different methods.
