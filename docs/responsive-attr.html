Responsive Attribute

Responsive attribute is an writeData("main_data['title']") Effect and provide responsive HTML attribute for adding HTML class to elements on different screen size. Classes written in responsive attribute will be defined by responsive suffix for different screen sizes. There are four different screen sizes defined by writeData("main_data['title']"), which are mbl for mobile screen size, tb for tablet screen size, dt for desktop and ldt for large desktop screen sizes. Classes defined in responsive attribute can be add to these four screen sizes either minimum or maximum screen size between these.

There are three different conditions for four different types of screen sizes. Every suffix of reponsive attribute screen sizes will be defined as Condition_Size. These conditions are

  • min for minimum screen size
  • max for maximum screen size
  • only for screen size only for defined Size i.e. both min and max conditions collectively.

Four different screen sizes are

  • mbl screen size minimum 0px and maximum 767px
  • tb screen size minimum 768px and maximum 991px
  • dt screen size minimum 991px and maximum 1239px
  • ldt screen size minimum 1240px.
