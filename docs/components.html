Hide Url on Existance

writeData("main_data['title']") provide an option to remove url of tag a if url is opened/active. Then url will be replaced by ##. This is provided in type ifme whose value is hideUrl. When hideUrl value will be added to ifme type then Url will be replaced if it is opened or active i.e. if url will be match to window current location.

Progress Bar

writeData("main_data['title']") provide progress bar which is very customizable, easy to use, with many options. Progress bar will be initiated with a class and some attributes and more classes for further options.

To initiate progress bar inside any element, progress-bar class will be added to class attribute. progress-bar default background color is transparent. Then to show progress value, another element inside progress-bar element will be added with class progress-value which will show progress value. Then to add progress value to progress bar, data-value attribute will be added to progress-value element whose value is between 0 and 100. progress-value default color is white.

Example

Options

Progress bar has certain options to show progress value, animated progress etc.

Show Progress value

To show progress value, progress-show class will be added to progress-value element.

Example

Animated Progress Bar

writeData("main_data['title']") provide animated progress bar. There are 2 different types of animated progress bars.

To add animated progress bar, add class progress-animated to progress-value element which will initiate progress animation, then add data-color with two different color which will splitted by (;). e.g. [data-color="red;white"]. There are no default color for animated progress bar, data-color attribute must be initiate to add colors to animated progress bar.

Example

To add 2nd animated progress bar, add class progress-animated_b instead of progress-animated This animation don't have any data-color attribute.

Animation Speed

There are 4 different types of animation speed for animated progress bar. These speed will be defined by adding class animation_speed-Speed-Value
There are 4 following Speed Values

  • slow
  • medium
  • fast
  • vfast
To defined any speed we will use these above Speed Values.
