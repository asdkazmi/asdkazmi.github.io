Hide Url on Existance

writeData("main_data['title']") provide an option to remove url of tag a if url is opened/active. Then url will be replaced by ##. This is provided in type ifme whose value is hideUrl. When hideUrl value will be added to ifme type then Url will be replaced if it is opened or active i.e. if url will be match to window current location.
